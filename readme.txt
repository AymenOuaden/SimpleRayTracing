----------------- Projet------------------------------

- Le projet est structuré en packages.

- J'ai utilisé une image de fond (elle s'affiche dans le ciel) pour mieux voir le reflet du ciel sur les sphères.(Idée inspirée de l'internet).
	* L'image du ciel se trouve dans le dossier "res", il doit se trouver dans la même hiérarchieque "JavaTga.java" et le paquet "RayTracing" (dans Src/).
	* On peut ignorer l'image du ciel, ==> On affiche le ciel avec une couleur Bleue ciel, il suffit d'utiliser le constructeur par défaut dans la classe "Sky.java".

- J'ai utilisé la classe Vec3f de Tps.

- Le code génère 3 scènes, deux scènes standard et une scène aléatoire. (Pour ajouter une scene il faut incrémenter le NB_SCENES dans la classe JavaTGA).

- Pour l'utilisation: copier tout le contenu de "src" dans un nouveau projet java. 


----------------- Résultat ----------------------------

- Les résultats générées sont dans le dossier résultat.
